## Max Callaghan

I'm a scientist at the Potsdam Institute for Climate Impact Research (PIK). I'm interested in using NLP to synthesise evidence on climate change.

Currently, I am leading work on evaluation for the [DESTINY](https://destiny-evidence.github.io/website/) project, where we aim to build tools that enable the responsible application of AI to scale up evidence synthesis.
